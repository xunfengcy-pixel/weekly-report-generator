# 📦 Mac 应用打包说明

## 方案一：一键打包脚本（推荐）

### 步骤

1. **将整个项目文件夹复制到 MacBook**

2. **打开终端，进入项目目录**
   ```bash
   cd ~/Documents/weekly_report_generator
   ```

3. **运行打包脚本**
   ```bash
   chmod +x build_mac_app.sh
   ./build_mac_app.sh
   ```

4. **等待打包完成**
   - 脚本会自动安装打包工具
   - 生成独立的 Mac 应用

5. **分发应用**
   - 打包完成后，`dist` 文件夹中包含可运行的应用
   - 将整个 `dist` 文件夹压缩后发给用户

---

## 方案二：手动打包

### 步骤

1. **在 MacBook 上安装依赖**
   ```bash
   pip3 install pyinstaller streamlit
   ```

2. **创建打包入口文件 `run_app.py`**
   ```python
   #!/usr/bin/env python3
   import os
   import sys
   import subprocess
   import webbrowser
   import time
   
   def main():
       print("🚀 启动周报生成器...")
       
       # 启动 Streamlit
       process = subprocess.Popen(
           [sys.executable, '-m', 'streamlit', 'run', 'app.py', 
            '--server.port=8501'],
           stdout=subprocess.PIPE,
           stderr=subprocess.PIPE
       )
       
       # 等待并打开浏览器
       time.sleep(3)
       webbrowser.open('http://localhost:8501')
       
       print("⚠️ 请不要关闭此窗口")
       process.wait()
   
   if __name__ == '__main__':
       main()
   ```

3. **执行打包命令**
   ```bash
   pyinstaller --onefile --add-data "app.py:." --add-data "config.py:." --add-data "template.html:." run_app.py --name "周报生成器"
   ```

4. **打包完成后**
   - 可执行文件在 `dist/周报生成器`
   - 将此文件发给用户即可

## 📤 用户使用方法

### 方式一：使用 .app 应用包（最推荐）

1. 下载 `周报生成器.zip` 并解压
2. 双击 `周报生成器.app`
3. 浏览器自动打开，开始使用

### 方式二：使用命令行版本

1. 下载并解压
2. 打开终端，进入文件夹
3. 运行：
   ```bash
   ./周报生成器
   ```
4. 浏览器自动打开

## ⚠️ 注意事项

### 首次运行权限问题

Mac 可能会提示"无法打开，因为无法验证开发者"，解决方法：

**方法 1：系统设置中允许**
1. 打开 系统设置 → 隐私与安全性
2. 找到"已阻止使用周报生成器"
3. 点击"仍要打开"

**方法 2：右键打开**
1. 按住 `Control` 键，点击应用
2. 选择"打开"
3. 在弹窗中点击"打开"

**方法 3：命令行解除限制**
```bash
xattr -cr /path/to/周报生成器.app
```

## 🔧 进阶：创建 DMG 安装包

如果需要更专业的分发方式，可以创建 DMG 安装包：

```bash
# 安装 create-dmg
brew install create-dmg

# 创建 DMG
create-dmg \
  --volname "周报生成器" \
  --window-pos 200 120 \
  --window-size 800 400 \
  --icon-size 100 \
  --app-drop-link 600 185 \
  "周报生成器.dmg" \
  "dist/周报生成器.app"
```
